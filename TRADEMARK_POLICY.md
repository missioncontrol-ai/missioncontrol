# MissionControl Trademark Policy

## Scope
This policy governs use of the names "MissionControl", "MissionControl MCP",
and project logos/wordmarks.

## What the Open Source License Covers
The Apache-2.0 license grants rights to code and documentation. It does not grant
rights to use project trademarks beyond fair and descriptive use.

## Permitted Use
You may:
- Refer to this project by name for factual, nominative purposes.
- State compatibility (for example, "works with MissionControl") when accurate.
- Link to the official repository and documentation.

## Restricted Use Without Written Permission
You may not, without prior written permission:
- Use MissionControl trademarks in your company/product/service name.
- Use logos/wordmarks in a way that implies endorsement or official status.
- Rebrand modified forks as "MissionControl" without clearly distinguishing them.
- Register confusingly similar domains, organizations, or package names.

## Forks and Derivative Distributions
Forks must:
- Keep Apache-2.0 notices intact.
- Clearly identify themselves as unofficial derivatives.
- Avoid using official marks as the primary product identity.

## Reporting Misuse
Report suspected misuse to the maintainers through `SECURITY.md` contact channels.
