MC TUI — Design Mockups

Static HTML layout prototypes. Open in browser, review design, then implement in Ratatui (Rust).
Views
  • Mission Matrix Main view — missions, klusters, task list with live progress, task detail pane default
  • Approval Queue Fast approve / deny / edit for gated tool calls with full context and reasoning key: a
  • Agent Feed Live event stream from all agents — phase, step, artifact, approval, system events key: f
  • Receipts Execution history — task receipts, artifact drill-down, tool call log, re-run key: q
Implementation Notes
  • TUI binary: mc tui or mc tui --mission <id>
  • Rust library: ratatui with crossterm backend
  • Data source: mc-server via MultiServerClient (SSE feed + REST)
  • Config: ~/.mc/config.toml for default mission, theme, refresh rate
