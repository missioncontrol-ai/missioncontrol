# Open Core Model

## Intent
MissionControl aims for broad open-source adoption while sustaining long-term
investment through optional paid offerings.

## Open Source Core (Apache-2.0)
Core includes:
- Mission/kluster/task data model and APIs
- MCP tool surface and local bridge components
- Baseline docs, tests, and integration examples

## Premium / Commercial Layer
Potential paid offerings may include:
- Hosted control plane and managed operations
- Enterprise-grade compliance and policy packs
- Advanced analytics, reliability tooling, and support SLAs

## Guardrails
- Core remains functional and genuinely useful without paid services.
- Premium features are additive, not lock-out of existing core behavior.
- Public documentation clearly distinguishes OSS vs premium capabilities.
